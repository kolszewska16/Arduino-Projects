# LightSensitiveRGBLamp
## Components
### Hardware
- Arduino UNO
- Resistors:
  - 330Ω (3x)
  - 10kΩ (1x)
- Photoresistor (LDR)
- RGB diode with common cathode
- Passive Piezo-Buzzer (HPM14AX)
### Software
- IDE: Arduino IDE
